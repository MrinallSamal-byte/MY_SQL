# 📘 MySQL Tutorial Repository

> A comprehensive, single-file MySQL tutorial covering database fundamentals through advanced techniques — designed for absolute beginners.

## 📖 About

This repository contains a complete, structured MySQL tutorial in **[TUTORIAL.md](TUTORIAL.md)**. It takes you from zero database knowledge to confidently designing schemas, writing efficient queries, managing security, and building real-world applications.

**Highlights:**
- 9 parts, 37 topics, 117+ SQL examples, 35+ practice exercises
- WHY / WHEN / HOW framework for every core topic
- Real-life analogies to make complex concepts intuitive
- 4 complete real-world projects (e-commerce, college system, sales analytics, traffic logging)
- Cheat sheet, glossary, and further-reading resources

## 🚀 Getting Started

### Prerequisites

- **MySQL 8.0+** installed on your machine (see [Part 2 in the tutorial](TUTORIAL.md#part-2-installation--setup) for step-by-step guides)
- A MySQL client — the built-in `mysql` CLI, [MySQL Workbench](https://dev.mysql.com/downloads/workbench/), or any GUI of your choice

### Quick Start

```bash
# Clone the repository
git clone https://github.com/MrinallSamal-byte/mysql.git
cd mysql

# Open the tutorial
# Use any Markdown viewer, VS Code, or read directly on GitHub
open TUTORIAL.md
```

Then follow along in **[TUTORIAL.md](TUTORIAL.md)**, running the SQL examples against your local MySQL instance.
